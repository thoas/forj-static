Comment ça se passe ?

Vous construisez votre tables sur-mesure avec l’interface de forj.shop, tous les éléments de sélection sont là, une gallerie d’exemples vous montre les résultats possibles pour fabriquer votre table sur-mesure. Une fois votre produit ajouté au panier et votre paiement effectué, nous reçevons la commande. Nous passons commande pour le bois et commençons la fabrication de votre table sur-mesure. Une fois la table terminée et testée, nous l’emballons, les pieds sont démontables pour faciliter le transport, nous vous livrons sous cinq semaines dans toute la France métropolitaine exepté la Corse.

Le prix de la livraison est inclus dans le prix ttc de la table sur-mesure, nous nous permettrons toutefois de le majorer en cas d’accès difficiles.
